# Audio Tools

このリポジトリには、オーディオ解析と信号生成に関する小さなプログラムが含まれています。

- `audio_analyzer`: 音声信号の歪や特性を解析するツール
- `audio_signal_generator`: 任意波形やテストトーンを生成するためのツール
